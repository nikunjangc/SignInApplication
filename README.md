# SignInApplication
SignIn application for 80 5th Avenue 
80 5TH Avenue building is a commercial building in NY. They are almost 800 visitors visiting every day to this building. And of which almost 400 visitors comes here once a month. This building has school offices, doctor’s clinics, law offices, medical labs, party venue, and other offices. 
The building management is strict about its security as there don’t want the wrong person to enter to their building. As they have to make every customer sign in every time, which in some scenario gives the wrong person to sign in and get inside the building. Space in the lobby is small which makes a queue of people waiting for their turn to sign in. A lot of customer gets angry as they have an important appointment, and they run late, as they have to sign in. 

Existing Solutions
* 	The tenants of those offices can print out the names of visitors every morning and give it to building management. Which will make easier for some old visitors but this will make problem to new visitors.
 
* 	 Even though visitor name is on the list they have to show their id and the security has to see the list, which will take the same amount of time.

Concept: 80 5TH Avenue  Sign in app

Visitors log is a mobile application, which helps visitors to sign in in easy manner. The major features include.

Application is develop for android device so can be run through any android device. When the visitors come to the building their will be a android tablet in a table, so the user has to enter the name and select the floor number, when the user enters the floor number it will give you a list of doctors/ lawyer/ psychiatric /therapist and many more in that current floor. So the user will select the desire name. And will enter the save button. When the user will look over the drop-down list to select the name of doctors/ lawyer/ psychiatric /therapist and many the in-build camera will take a picture. 

In the back End all the save data will be stored in a hash map data structure with the in-built time.
When opening the application there will be a login screen and the user will have a option to see the log or sign in mode. In log mode user can see the information of all the visitors, which is sorted in the increasing time, and has a search option to search for the visitor by floor or the tenant/member name.
